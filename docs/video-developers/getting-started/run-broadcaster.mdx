# Run Broadcaster

Starting livepeer with the `-broadcaster` starts the node in broadcaster mode
enabling you to stream video into the Livepeer network.

```bash
livepeer \
  -network rinkeby \
  -ethUrl <ETH_URL> \
  -broadcaster \
  -currentManifest
```
